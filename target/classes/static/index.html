📡 API 端点
GET /api/health 健康检查

检查服务状态、版本、支持的方言和功能列表。

POST /api/sql/parse 解析 SQL 语法树

支持 MySQL / PostgreSQL / SparkSQL。返回格式化的语法树文本。

POST /api/sql/validate 验证 SQL 语法

检查 SQL 是否语法正确。

POST /api/sql/metadata 提取 SQL 元数据（表/列/别名/函数）

使用 Visitor 模式提取 MySQL SQL 的表名、列名、别名、函数调用等元数据。

POST /api/sql/audit SQL 审计分析（风险检测/表访问追踪）

使用 Listener 模式审计 SQL：危险操作检测、表访问追踪、优化建议等。

POST /api/sql/format SQL 格式化/美化

将凌乱的 SQL 格式化为标准缩进格式，支持自定义缩进和关键字大小写。

POST /api/sql/lineage 列级数据血缘分析 (JSON)

分析 SQL 的列级数据血缘（Data Lineage），返回完整的血缘图 JSON。

POST /api/sql/lineage/html 血缘图交互式 HTML 可视化

返回自包含的交互式 HTML 血缘图页面，支持 hover 高亮、点击追踪。

点击「在新窗口查看」将在新标签页打开 HTML 血缘图
POST /api/sql/lineage/ascii 血缘图 ASCII 文本报告
POST /api/sql/visualize/ascii 语法树 ASCII 文本
